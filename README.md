# HIMTI Competition Landing Page

Landing page statis HIMTI Competition menggunakan HTML, CSS, JavaScript.

Project ini menggunakan konsep HTML partials agar tiap section bisa diedit terpisah.

---

## Tech Stack
- HTML
- CSS
- JavaScrip

---

## Folder Structure
- index.html
- partials/
  - header.html
  - hero.html
  - competitions.html
  - about.html
  - contact.html
  - footer.html
- js/
  - include.js
- css/
  - main.css
  - base/
  - components/
  - sections/
- assets/
  - icons/
  - images/

---

## How to Run (Local with VSCode)
1. Install extension "Live Server" pada VSCode
2. Klik kanan `index.html` -> "Open with Live Server"

---

## Notes
- Hindari nama file pakai spasi. Kalau ada, pastiin path pakai `%20` atau rename jadi kebab-case.
- CSS pakai variable di `css/base/variables.css` untuk warna dan font.
- Pastikan `css/main.css` import file sesuai urutan: base -> components -> sections.
